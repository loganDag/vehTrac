## How to contribute
First off, thank you for wanting to contribute to this project and may God bless you and keep you. There are many ways to contribute to this project, you can fork, copy, or clone it and make changes yourself. If you do the following, please make and issue and then assign yourself in order for others to see that it is being workked on and to push it.


## Contributing to Development 

If you want to contribute to development, you can clone or fork this repo and makes the changes yourself, I do ask please make an "Issue" first with the appropriate tags and then do so from there so it can be easily tracked. I also ask to keep development right now to HTML 5, CSS, BootStrap, JavaScript, PHP 8, and MySql.


## Feedback
If you want to contribute via feedback on current versions or other things, i.e. current code or development, please also either make an issue or bring it up in the discussions as well either here or on Discord.

## Documentation
Contributing to documentation of current versions their new features (if any) and what was changed. Docs can be found under the docs folder and the Wiki Page for now. If you want to contribute to the docs, please use the Documentation in the "Docs" folder, once reviewed and approved, it will be published to the wiki and any other related page. Thank you!
